## Installation
#### From the repository:
* To existing pixi env:
    ```bash
    pixi add --git https://github.com/cda24/pirate pirate --pypi  
    ```

* With pip:
    ```bash
    pip install -e git+https://github.com/cda24/pirate
    ```
#### From source:
1. Clone the repository:

    ```bash
    git clone https://github.com/cda24/pirate.git 
    ```

2. Install the package:

    ```bash
    pip install -e .
    ```
